# Субмодули ftrack_inout

| Субмодуль | Описание |
|-----------|----------|
| **browser** | Браузер ассетов, перенос файлов, DCC-интеграция (Houdini, Maya) |
| **input** | Finput: загрузка компонентов ассетов, DCC-агностичное ядро и адаптеры |
| **publisher** | Публикация версий и компонентов в ftrack |
| **common** | Сессия ftrack, кэш, общие утилиты, работа с путями |
| **ftrack_hou_utils** | Утилиты для Houdini (API-клиент, ftrack-хелперы) |
| **asset_watcher** | Слежение за ассетами, уведомления DCC об обновлениях, watch/unwatch |
